# À propos

Cette application a été créée pour illustrer comment construire un chatbot complet avec Streamlit et l'API OpenAI. Elle met en avant une interface simple, personnalisable et prête pour le déploiement sur Streamlit Cloud.

## Technologies
- [Streamlit](https://streamlit.io/)
- [OpenAI Python SDK](https://github.com/openai/openai-python)

## Auteur
Projet généré automatiquement par un assistant IA.
